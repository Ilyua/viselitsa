#Program "Viselitsa"

>What is it?

It is simple console game. Try to open all letters in the word
You may did 7 mistakes



>How to use?

You need to install Ruby interpreter (version 2.3+)

Then open your terminal and type follow instructions:

```ruby viselitsa.rb```

Press `Enter`

Enjoy!
